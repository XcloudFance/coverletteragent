# coverletteragent
A multi-agent sequential flow to generate cover letter based on your resume + the targeted background

## Usage
Go to `main.py` file and edit your own api keys.

Install the env by:
```bash
pip install gradio crewai langchain-community duckduckgo-search arxiv PyPDF2 requests
```

Then you can start running gradio by:
```bash
python main.py
```
